# Plug Your Build

The indie maker directory where backlinks compound. Submit your SaaS, newsletter, course, Gumroad asset, Etsy storefront, Discord server, podcast, productized service, or weekend side project. Permanent listings, human reviewed, with a dofollow backlink to your site.

## What it is

Plug Your Build is a categorized directory built specifically for indie makers and solo operators. Every listing is permanent and persists as long as the badge stays embedded (free tier) or the subscription stays active (paid tiers). The site has twelve top-level categories, browse pages for each, click and impression analytics on the maker dashboard, an Editor's Pick spotlight, an auto-rotating "In the Spotlight" rail of paid placements, and a Sunday newsletter that surfaces the week's best indie work to a growing list of indie-curious readers.

The directory exists because indie products lose visibility quickly on social media but retain value indefinitely on directories. A tweet has a half-life of about eighteen minutes. A Product Hunt launch is a twenty-four-hour sprint. A directory listing produces a steady drip of search-qualified traffic for as long as it stays live.

## The backlink flywheel

Plug Your Build runs a true backlink flywheel. Every listing gets a permanent dofollow link from plugyourbuild.com to your site. As the directory's domain authority climbs, the link gets more valuable. The free tier is the engine: free listings trade a "Listed on Plug Your Build" badge embedded on the maker's own site or GitHub README for a permanent dofollow link back. Every embedded badge sends a signal to plugyourbuild.com, which lifts the value of every paid listing's outbound link, which makes paid listings more valuable, which funds the directory. The trade works in both directions.

## Why a directory rather than a launch platform

Different jobs. A launch platform optimizes for the day-of spike. A directory optimizes for the long tail.

For most indie products, the directory does more total work over twelve months than any individual launch. A cluster of five quality directory listings, maintained and updated, can drive a thousand-plus monthly targeted visitors by month twelve, with no ongoing effort beyond keeping the listings current. The traffic from directories is search-qualified in a way social traffic almost never is, because directory browsers searched their way to the listing.

Plug Your Build is one such directory, built specifically for indie work rather than enterprise SaaS or AI hype cycles.

## What's allowed

Real indie work in any of the categories above. The bar is intentionally low for indie and solo work. The moderation focus is on quality and safety, not gatekeeping.

Concrete examples of submissions that fit:

- A solo-built SaaS at $0 MRR
- A weekly newsletter with twelve subscribers
- A Discord server with fifty members
- A Gumroad template doing $30 a month
- A productized service one person delivers
- A weekend side project that's still in beta
- A podcast with eight episodes published
- An open-source devtool or CLI utility

Stage is not a gate. Polished and rough listings sit next to each other in browse, sorted by recency and category.

## What's not allowed

SFW only. Plug Your Build is a clean, family-safe directory across every category. The following are immediate rejection:

- NSFW content of any kind (adult content, explicit imagery, sexual services or platforms)
- Scams, MLM, get-rich-quick schemes, pyramid structures
- Gambling, betting, casino-adjacent products
- Crypto pump-and-dump tokens, rug-pull projects, "investment opportunities" without substance
- Illegal goods, controlled substances, weapons
- Misleading, deceptive, or impersonation listings
- Spam, duplicate submissions, or low-effort listings created in bulk
- Stolen IP or counterfeit goods
- Hate speech, harassment, or products targeting groups for harm

Borderline cases are reviewed individually. The standard: would a typical professional be comfortable having this listing surface on their work computer next to other indie products. If the answer is no, the listing is rejected with a proration refund.

## How submission works

### Free submission

1. Sign up and pick the Free tier at [plugyourbuild.com/dashboard/listings/new](https://plugyourbuild.com/dashboard/listings/new).
2. Fill in basics, upload a square logo, pick a badge host (your website or a public GitHub repo).
3. Paste the embed code into your site footer or GitHub README and click Verify.
4. The listing enters a 3-day queue. Most submissions are manually approved sooner. If we don't reject it within 3 days, it auto-publishes.
5. Once live, the badge is checked weekly. Two consecutive missed checks pauses the listing until the badge is re-embedded.

### Paid submission

1. Pick Indie, Pro Lifetime, or Spotlight at [plugyourbuild.com/pricing](https://plugyourbuild.com/pricing).
2. Fill in the same details. No badge required, no queue.
3. Stripe handles billing. Cancel anytime through the Stripe Customer Portal.
4. A human reviewer checks the submission for SFW compliance and category fit. Most submissions go live within 24 hours.

If a submission needs an edit before approval, you receive an email with a specific note. If a paid submission is rejected outright, the Stripe subscription is canceled with a proration refund.

## Listed on Plug Your Build · Embed the badge

Every listing on PYB can embed a "Listed on Plug Your Build" badge on the maker's own site or GitHub README. Free-tier listings require it. Paid-tier listings get it as a bonus social-proof asset.

The badge endpoint serves three style variants:

```
https://plugyourbuild.com/api/badge/your-slug?style=dark
https://plugyourbuild.com/api/badge/your-slug?style=light
https://plugyourbuild.com/api/badge/your-slug?style=transparent
```

HTML embed for a website footer:

```html
<a href="https://plugyourbuild.com/listing/your-slug" rel="dofollow">
  <img src="https://plugyourbuild.com/api/badge/your-slug?style=dark"
       alt="Listed on Plug Your Build" width="180" height="40" />
</a>
```

Markdown embed for a GitHub README:

```markdown
[![Listed on Plug Your Build](https://plugyourbuild.com/api/badge/your-slug?style=dark)](https://plugyourbuild.com/listing/your-slug)
```

The badge does two things at once. It signals to other indie makers that the product is part of a curated directory, and it produces a small reciprocal SEO benefit for both sides. As PYB's domain authority climbs, the value of every embedded badge climbs with it.
